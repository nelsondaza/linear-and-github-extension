and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Security

- Documented security best practices for API key management

## [0.0.4] - 2025-12-11

### Current Features

- Linear issue detection on GitHub pull request pages
- Linear issue detection on GitHub compare views
- Display of Linear issue details (status, priority, assignee, estimates)
- Beautiful UI integration with GitHub's design language
- Secure local storage of Linear API keys
- Support for Chrome browser (Manifest V3)

## Release Notes

### Version 0.0.4

This is the current stable release of LAGE. It includes:

- Core functionality for linking Linear issues with GitHub pull requests
- Secure API key storage
- Clean UI integration
- Support for Chrome browser
